#  Sales Data Analysis Project

#  Detailed overview
This project analyzes product-wise business revenue using Python.  
It performs basic statistical analysis and visualizes the data using graphs.

#  Programming languages Used
- Python
- Pandas
- Matplotlib

#  Features
- Data analysis (Total, Average, Max, Min)
- Bar chart visualization
- Line graph with trend
- High and low performance identification

#  Output
- Revenue comparison using bar chart
- Trend analysis using line graph
- Average revenue line for better insights

#  Conclusion
This project helps in understanding business performance and identifying top and low-performing products using data visualization techniques.
